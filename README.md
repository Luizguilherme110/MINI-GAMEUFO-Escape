# UFO Escape

Mini game em HTML, CSS e JavaScript inspirado no jogo do dinossauro, mas com um OVNI desviando de meteoritos.

## Tecnologias
- **HTML** para a estrutura da página.
- **CSS** para o visual do layout e fundo espacial.
- **JavaScript (Canvas)** para o loop do jogo, física, colisões e efeitos.

## Como jogar
- Use **↑** e **↓** para mover o OVNI.
- Pressione **Espaço** para atirar.
- Desvie dos meteoritos e derrote o boss quando ele aparecer.
- O score aumenta conforme você sobrevive.

## Executar localmente
Basta abrir o arquivo `index.html` no navegador. Ou acesse: https://luizguilherme110.github.io/MINI-GAMEUFO-Escape/
